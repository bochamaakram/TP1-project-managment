# TP1 - Projet Collaboratif GitHub

## Description du projet
Ce projet est un exercice pratique (TP1) visant à maîtriser les fondamentaux de la collaboration sur GitHub. Il consiste à créer un mini-projet web, le versionner, collaborer via des branches et des Pull Requests, et le déployer.

L'objectif est de se familiariser avec :
- L'initialisation d'un dépôt Git
- La gestion des branches et des workflows collaboratifs
- La publication sur GitHub Pages (ou autre)

## Technologies utilisées
- **HTML5 / CSS3** (Structure et mise en page)
- **JavaScript** (Logique client)
- **Git & GitHub** (Versionning et collaboration)

## Instructions d’installation

1. Cloner le projet :
   ```bash
   git clone https://github.com/akram/tp1-github-collab-[votre-nom].git
   ```

2. Accéder au dossier du projet :
   ```bash
   cd tp1-github-collab-[votre-nom]
   ```

3. Lancer le projet :
   - Ouvrez simplement le fichier `index.html` dans votre navigateur.
   - (Si version Node.js) : `npm install` puis `npm start`.
